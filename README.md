# Community Bread Order App

Mobile-friendly bread ordering app for a WhatsApp/community group.

## Files

Keep these files together in one folder:

- index.html
- styles.css
- app.js
- firebase-config.js
- manifest.json
- sw.js
- README.md

## Setup summary

1. Create a Firebase project.
2. Add a Web App in Firebase.
3. Enable Anonymous Authentication.
4. Create Realtime Database.
5. Copy Firebase config into firebase-config.js.
6. Upload all files to a GitHub repository.
7. Enable GitHub Pages from the main branch and root folder.

Default admin PIN is in firebase-config.js:

```js
window.ADMIN_PIN = "1234";
```

Change it before sharing the app.
